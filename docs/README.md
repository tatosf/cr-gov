# Documentación de GobiernoCR

Guías para contribuir al proyecto. Si es tu primera vez, empieza por el [CONTRIBUTING.md](../CONTRIBUTING.md) en la raíz del repositorio para configurar tu entorno, y luego usa estos documentos para entender cómo funciona el proyecto y dónde puedes aportar.

## Índice

- **[Arquitectura](./arquitectura.md)** — Visión general del stack, del flujo de datos y de cómo se organizan las carpetas del proyecto.
- **[Sistema de scraping dinámico](./sistema-de-scraping.md)** — Propuesta completa para migrar los datos semilla estáticos a un sistema dinámico basado en Cloudflare Workers, D1, Cron Triggers y Browser Rendering. **Este es el proyecto prioritario y la mejor forma de contribuir a largo plazo.**
- **[Roadmap de contribuciones](./roadmap-contribuciones.md)** — Lista viva de áreas donde el proyecto necesita ayuda, organizada por dificultad y por tipo (datos, visualizaciones, UI/UX, infraestructura).
- **[Agregar una fuente de datos](./agregar-fuente-de-datos.md)** — Guía paso a paso para integrar una nueva fuente gubernamental al sistema de scraping.

## ¿Cómo funcionan estas guías?

Cada documento es una "wiki" independiente pensada para que cualquier persona pueda tomar el tema, abrir un issue, y arrancar a trabajar sin depender del equipo mantenedor. Si encuentras un error o ves algo que se puede mejorar, abre un PR — los docs también son contribuibles.
